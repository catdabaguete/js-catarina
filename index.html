Equilibro Quimico

  • equilibrio químico é. um fenômeno que ocorre em reações químicas reversíveis, onde a velocidade da reação direta (formação de produtos) é igual.à velocidade da reação nversa (formação.de reagentes)
  • Condições para o equilíbrio químico ◦ Sistema fechado: Não há troca de matéria com 0 ambiente. Temperatura constante: A temperatura do sistema deve ser mantida constante
  • Fatores que afetam o equilíbrio químico: Concentração: Alterações na concentração dos reagentes ou produtos podem deslocar o equilíbrio. Temperatura: Mudanças na temperatura podem afetar a posição do equilíbrio. Pressão: Em sistemas gasosos, mudanças na pressão podem deslocar o equilíbrio. Catalisadores: Podem acelerar a velocidade com que o equilíbrio é atingido, mas não alteram a posição do equilíbrio
  • equilibrio quimico constante A constante de equilíbrio, ( K ), é uma medida da posição do equilíbrio e é definida em termos das concentrações dos reagentes e produtos. Para a reação genérica: aA+bB⇌cC+dD A constante de equilíbrio é expressa como: Kc​=[A]a[B]b[C]c[D]d​ Onde ( [A] ), ( [B] ), ( [C] ) e ( [D] ) são as concentrações molares das substâncias respectivas, e ( a ), ( b ), ( c ) e ( d ) são seus coeficientes estequiométricos
